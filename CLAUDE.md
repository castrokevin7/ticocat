## Architecture Overview

**TiCoCat** is a bilingual (Spanish/Catalan) React SPA for a community association, with AWS Amplify backend for data and authentication.

### Two Main Sections

1. **Informative** (`/es/*` or `/cat/*`): Public landing page with events, benefits, FAQs, and associate verification
2. **Social Network** (`/es/social/*` or `/cat/social/*`): Authenticated member area with community listing, profiles, and resources

### Key Directories

- `src/views/informative/` - Public pages (main landing, events, benefits, FAQs)
- `src/views/social-network/` - Authenticated member pages (community, profile, resources)
- `src/components/` - Reusable UI components (Cards, MK* wrappers, Navbars, Footers)
- `src/ui-components/` - Auto-generated Amplify Studio components (do not edit manually)
- `src/models/` - Auto-generated DataStore models (do not edit manually)
- `src/assets/theme/` - MUI theme customization (colors, typography, component overrides)
- `src/utils/` - Utilities (Translator, date formatting, auth actions)
- `amplify/backend/api/TICOCAT/schema.graphql` - GraphQL data model definition

### Bilingual System

- Language detected from URL pathname (`/es/` vs `/cat/`)
- `src/utils/Translator.js` + `src/utils/Translations.json` handle translations
- GraphQL models have dual fields (e.g., `title`/`title_cat`, `description`/`description_cat`)

### Data Models (GraphQL/DataStore)

- **FAQ**: Questions with bilingual content and links
- **Benefit**: Provider benefits with contact info and media
- **Event**: Community events with dates, galleries, locations
- **Associate**: Members with roles (BoardPosition enum), interests (Interests enum), profile settings

### Component Patterns

- **MK* components** (`MKBox`, `MKTypography`, `MKButton`, etc.): Custom wrappers around MUI components with app-specific styling
- Theme helpers in `src/assets/theme/functions/`: `boxShadow()`, `hexToRgb()`, `linearGradient()`, `pxToRem()`, `rgba()`

### AWS Amplify Integration

- Authentication via Cognito (`Authenticator.Provider` wraps app)
- DataStore for offline-first data sync
- `src/aws-exports.js` contains Amplify configuration (generated)

## Important Notes

- Files in `src/models/` and `src/ui-components/` are auto-generated by Amplify - do not edit directly
- ESLint config excludes `src/models/` from linting
- The app uses React Router v6 with language-prefixed parallel routes
